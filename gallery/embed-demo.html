Framed same-origin content

This page is loaded inside a sandboxed iframe by the gallery's Embed route. It is served same-origin, so the gallery stays checker-clean while still demonstrating the isolation boundary. A real deployment would point the frame at a foreign network surface (a map, a dashboard, an OAuth page); those external cases live in the test fixtures, never in the shipped gallery.

A navigation link inside the frame stays inside the frame's own browsing context.
